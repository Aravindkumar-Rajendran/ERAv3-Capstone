# Frontend Project

This project is a React application structured to provide a clean and efficient development experience.

## Project Structure

- **src/**: Contains the source code for the application.
  - **components/**: Contains reusable components.
  - **pages/**: Contains the main pages of the application.
  - **styles/**: Contains CSS styles for the application.
  - **types/**: Contains TypeScript interfaces and types.
  - **App.tsx**: The main application component that sets up routing and layout.

- **public/**: Contains static files.
  - **index.html**: The main HTML file for the application.

- **package.json**: Configuration file for npm, listing dependencies and scripts.

- **tsconfig.json**: TypeScript configuration file.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd frontend
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

## Usage

After starting the development server, open your browser and navigate to `http://localhost:3000` to view the application.
